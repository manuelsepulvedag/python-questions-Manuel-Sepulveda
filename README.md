## Preguntas del Laboratorio

# 1. Reconstrucción de Atractores con Lag-Embeddings
**Misión:** Implementar una función que transforme una serie temporal escalar en un espacio de fases multidimensional utilizando retardos (*lags*) y reduzca su dimensionalidad mediante PCA para capturar la dinámica del sistema.

# 2. Optimización del Umbral de Decisión vía Cost-Benefit Matrix
**Misión:** Desarrollar un algoritmo que encuentre el umbral de probabilidad óptimo para minimizar el impacto económico de los errores, asignando pesos personalizados a los Falsos Negativos y Falsos Positivos.

# 3. Filtrado de Ruido en Señales con EMA Adaptativa
**Misión:** Crear un filtro de Media Móvil Exponencial cuya agresividad de suavizado se ajuste dinámicamente en tiempo real en función de la volatilidad detectada en la señal del sensor.

# 4. Evaluación de Estabilidad de Clusters mediante Bootstrapping
**Misión:** Determinar si los grupos encontrados por un modelo de clustering son consistentes y reales mediante técnicas de remuestreo (bootstrap) y el cálculo del Índice Rand Ajustado (ARI).
